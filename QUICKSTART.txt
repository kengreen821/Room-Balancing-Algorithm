QUICK START GUIDE
=================

1. OPEN THE APP
   - Double-click "index.html" to open in your browser

2. LOAD DATA
   - Click "Load Reservations JSON"
   - Select "embassy_suites_reservations_final.json"

3. TEST WITH A SOLD-OUT DATE
   - Select: 2026-01-17 (100% occupancy - 321 arrivals)
   - Click "Analyze Arrivals"

4. REVIEW ALERTS
   - You'll see overbookings for TDBN, KSVN, and ADA rooms
   - Check each alert box after "handling" it
   - Notice: Low-priority rates (Third-Party, Hilton Go) get upgraded first

5. FINALIZE
   - Once all boxes checked, click "Finalize All Room Assignments"
   - See all 321 assignments
   - Export to CSV if needed

GITHUB SETUP:
-------------
1. Create new repository: "hotel-room-balancer"
2. Upload these files:
   - index.html
   - room-balancer.js
   - README.md
   - embassy_suites_reservations_final.json (sample data)

3. Enable GitHub Pages (Settings → Pages → Deploy from main branch)
4. Your app will be live at: https://[username].github.io/hotel-room-balancer
