CHANGE 1: Added In House and Due Outs Columns
----------------------------------------------

The Overbooking Analysis table now shows:

┌──────────────┬──────────┬──────────┬──────────┬───────────┬────────┬──────────────┐
│ Room Type    │ In House │ Due Outs │ Arrivals │ Available │ Status │ Overbooked By│
├──────────────┼──────────┼──────────┼──────────┼───────────┼────────┼──────────────┤

│ TDBN         │   110    │    15    │   119    │    31     │   OK   │      -       │
│ KEXN         │     2    │     0    │     4    │     1     │  OVER  │      3       │
├──────────────┼──────────┼──────────┼──────────┼───────────┼────────┼──────────────┤
│ TOTALS       │   197    │    27    │   253    │   151     │  OVER  │      3       │
└──────────────┴──────────┴──────────┴──────────┴───────────┴────────┴──────────────┘

**Definitions:**
- **In House**: Guests staying over from previous night(s)
- **Due Outs**: Guests checking out TODAY
- **Arrivals**: Guests checking in TODAY (what was previously "Booked")
- **Available**: Inventory - (In House - Due Outs)

**Formula:**
Available for new arrivals = Total Inventory - (In House - Due Outs)

Example:
- KNGN has 94 total rooms
- 85 guests staying over (In House)
- 12 checking out today (Due Outs)
- Net occupied: 85 - 12 = 73
- Available: 94 - 73 = 21 rooms
- 91 arrivals vs 21 available = Would be overbooked, but algorithm upgrades to KSVN/KEXN


CHANGE 2: Realistic Room Type Distribution

**NEW Distribution** (Realistic):
- KNGN: 45% ← MOST COMMON (Standard King)
- TDBN: 35% ← VERY COMMON (Standard Double)
- KSVN: 8% (Park View King)
- TSVN: 5% (Park View Double)
- NKSQA: 3% (ADA King)
- KEXN: 2% (Executive King)
- Other: 2% (Premium suites, etc.)

**Why this matters:**
- 80% of bookings are now KNGN or TDBN (standard rooms)
- Premium rooms (KEXN, NKSP, suites) are RARE - as they should be
- Matches real-world booking patterns
- Makes overbooking scenarios more realistic

**Impact on Algorithm:**
- Most guests book KNGN or TDBN
- When overbooked, upgrade paths focus on these common types
- Premium upgrades (to KEXN, NKSP) are special events, not routine
- Named suites (KSPN, KOTN, KSLN) are emergency-only
